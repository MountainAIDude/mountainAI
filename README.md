# Mountain AI Site

This is a static site. The repo should include the following files so the site renders correctly and previews well on social/search.

## Required files
- `index.html`: Single-page site with styles and metadata.
- `mountainAIepic2.jpg`: Hero background image (used by Open Graph/Twitter preview).
- `mountainAIcompute.jpg`: Middle section background image.
- `mountainAImine3.jpg`: Underground section background image.
- `mountainfavicon.png`: Primary icon and touch icon.
- `favicon.ico`: Favicon fallback for browsers/search results.
- `safari-pinned-tab.svg`: Safari pinned-tab icon.

## Optional files
- `CNAME`: Custom domain for GitHub Pages (if using Pages with a custom domain).

## Notes
- Open Graph/Twitter meta tags in `index.html` use absolute URLs pointing to `https://mountainai.ca/`.
- If you change the domain or file names, update the meta tags in `index.html`.
